# 米哈游亮色设计系统
# Hoyoverse Light Theme Design System - 「光之城」

## 设计理念

融合《原神》装备页的精致毛玻璃与《绝区零》的现代潮酷感，打造游戏主页风格的亮色主题。

---

## 1. 全局背景系统

### 核心背景
- **基础色**: 柔和暖白 `#f8f7f4`
- **装饰光晕**: 顶部珊瑚橙、右下青蓝、左侧金色
- **网格纹理**: 微妙的浅色网格线
- **浮动球体**: 青蓝色装饰元素，缓慢漂浮

### 氛围营造
```css
background:
  /* 顶部暖色光晕 */
  radial-gradient(ellipse 80% 50% at 50% -10%, rgba(255, 122, 92, 0.12), transparent 50%),
  /* 右下青蓝光晕 */
  radial-gradient(ellipse 60% 40% at 100% 100%, rgba(0, 200, 180, 0.08), transparent 50%),
  /* 左侧金色点缀 */
  radial-gradient(ellipse 40% 60% at 0% 50%, rgba(212, 168, 83, 0.06), transparent 40%),
  #f8f7f4;
```

---

## 2. 卡片与层级系统

### 材质特性
- **毛玻璃**: `backdrop-filter: blur(16px) saturate(150%)`
- **半透明背景**: `rgba(255, 255, 255, 0.65)`
- **悬停增强**: 背景加深 + 发光边框显现
- **内部光泽**: 顶部渐变高光层

### 边框发光（悬停触发）
```css
/* 珊瑚橙到青蓝的渐变边框 */
background: linear-gradient(135deg,
  rgba(255, 122, 92, 0.4) 0%,
  transparent 25%,
  transparent 75%,
  rgba(0, 200, 180, 0.3) 100%);
```

### 斜切角装饰
- 卡片头部左上角有珊瑚橙三角形装饰
- 整体采用圆角矩形（保留柔和感）

## 3. 色彩系统

### 主色调 - 珊瑚橙（原神暖调）
| 名称 | 色值 | 用途 |
|------|------|------|
| `--coral` | `#ff7a5c` | 主强调/CTA/激活态 |
| `--coral-light` | `#ff9a7a` | 悬停态 |
| `--coral-dark` | `#e85a3c` | 按下态 |

### 辅色 - 青蓝（ZZZ科技感）
| 名称 | 色值 | 用途 |
|------|------|------|
| `--cyan-accent` | `#00c8b4` | 链接/代码/状态指示 |
| `--cyan-light` | `#40e8d4` | 悬停态 |
| `--cyan-dark` | `#00a894` | 按下态 |

### 金色系（原神精致感）
| 名称 | 色值 | 用途 |
|------|------|------|
| `--gold-warm` | `#d4a853` | 装饰/边框点缀 |
| `--gold-light` | `#f0c86a` | 高亮装饰 |

### 渐变
```css
--gradient-primary: linear-gradient(135deg, #ff7a5c, #ff9a7a, #ffb090);
--gradient-cyan: linear-gradient(135deg, #00c8b4, #40e8d4);
--gradient-gold: linear-gradient(135deg, #d4a853, #f0c86a);
```

## 4. 字体系统

### 标题字体 - ZZZ潮酷风格
- **首选**: "Orbitron" - 科技感、粗壮扁平
- **备选**: "Rajdhani" - 现代潮酷
- **特点**: 粗字重 (500-700)，紧凑字距，大写字母

### 正文字体 - 原神优雅风格
- **首选**: "Noto Sans SC" - 中文黑体
- **英文**: "DM Sans Variable"
- **特点**: 干净、优雅、高可读性

/* 基础阴影 */
--shadow-md: 0 4px 16px rgba(45, 42, 38, 0.08), 0 2px 4px rgba(45, 42, 38, 0.04);

--shadow-lg: 0 8px 32px rgba(45, 42, 38, 0.1), 0 4px 8px rgba(45, 42, 38, 0.06);

## 6. 动效系统

### 时间曲线
- **快速**: `150ms cubic-bezier(0.22, 1, 0.36, 1)`
- **标准**: `250ms cubic-bezier(0.22, 1, 0.36, 1)`
- **弹性**: `400ms cubic-bezier(0.34, 1.56, 0.64, 1)`

### 关键动效
```css
/* 状态指示器脉冲 */
@keyframes pulse-glow {
  0%, 100% { opacity: 1; box-shadow: 0 0 12px rgba(0, 200, 180, 0.5); }
  50% { opacity: 0.8; box-shadow: 0 0 20px rgba(0, 200, 180, 0.7); }
}

/* 浮动装饰球 */
@keyframes float-orb {
  0%, 100% { transform: translate(0, 0) scale(1); }
  25% { transform: translate(-30px, 20px) scale(1.05); }
  50% { transform: translate(-10px, -20px) scale(0.95); }
  75% { transform: translate(20px, 10px) scale(1.02); }
}
```

## 7. 组件规范

### 按钮
- **主按钮**: 珊瑚橙渐变背景，白色文字，发光阴影
- **次要按钮**: 透明背景，浅色背景悬停
- **复制按钮**: 青蓝色边框，图标+文字

### 标签/Tab
- **激活态**: 珊瑚橙渐变背景，白色文字，上移+发光
- **未激活**: 透明背景，灰色文字

### 卡片
- **容器**: 毛玻璃材质，圆角矩形
- **悬停**: 背景加深，发光边框显现，轻微上移
- **装饰**: 左上角三角形、标题竖线装饰

### 状态指示器
- 青蓝色圆点，脉冲发光动画
- 配合模型指示器使用
